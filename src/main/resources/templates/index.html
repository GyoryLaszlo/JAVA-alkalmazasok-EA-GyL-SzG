Gyakori kérdések

SOAP alapú devizaárfolyam és FOREX alkalmazás

Az alkalmazás a Magyar Nemzeti Bank (MNB) nyilvános SOAP webszolgáltatásán keresztül kérdezi le a legfrissebb devizaárfolyamokat. A válasz XML formátumban érkezik, amit a rendszer feldolgoz, majd táblázatosan és grafikusan megjelenít.

A FOREX modul egy oktatási célú szimuláció, ahol a felhasználó virtuálisan „kereskedhet” különböző devizapárokkal (pl. EUR/USD, GBP/HUF). A cél, hogy bemutassa, hogyan működnek az árfolyammozgások és a piaci döntések hatása.

A rendszer Java Spring Boot keretrendszerre épül, a megjelenítéshez pedig Thymeleaf sablonmotor és Bootstrap alapú reszponzív téma használatos. Az interaktív grafikonokat a Chart.js könyvtár biztosítja.

Az MNB SOAP API biztosítja a hivatalos napi devizaárfolyamokat több mint 25 devizára, beleértve az EUR, USD, GBP, CHF és HUF árfolyamokat, valamint historikus adatokat is megadott időintervallumra.

Az alkalmazás oktatási célokat szolgál: bemutatja a webszolgáltatások (SOAP) integrálását, a Spring Boot architektúra használatát, valamint a front-end és back-end kommunikációt egy modern webalkalmazásban.
